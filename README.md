# nt8-autologin
Very quick and dirty app to automatically launch and log into NinjaTrader 8 in non-multi-provider mode.
More to come. Probably.

## Environment Vars

* NT8A_PATH - path to NinjaTrader.exe
* NT8A_USER - NT username
* NT8A_PASS - NT password
* NT8A_LIVE - TRUE for live trading, FALSE for sim
